# Assets Folder

## DataBase.csv

This folder contains the CSV database file used by the Medical Exemption Plugin.

### CSV File Structure

The `DataBase.csv` file should have the following columns (10 columns total):

| Column Index | Column Name | Description |
|--------------|-------------|-------------|
| 0 | Header | Header/ID field |
| 1 | Priority | Selection priority (اولویت انتخاب) |
| 2 | Year | Year of exemption (سال) |
| 3 | Section Name | Name of section (نام بخش) |
| 4 | Section Code | Code of section/article (کد ماده) |
| 5 | (unused) | Reserved column |
| 6 | Subject | Subject/topic (موضوع) |
| 7 | Article | Article number (بند) |
| 8 | Summary | Brief summary (خلاصه) |
| 9 | Description | Detailed description (شرح تفصیلی) |

### Important Notes

1. The first row should be the header row (it will be skipped during processing)
2. Replace the sample data in `DataBase.csv` with your actual medical exemption data
3. Ensure all rows have 10 columns (even if some are empty)
4. Use UTF-8 encoding for Persian text

### Build Folder

The `/build/` subfolder will be automatically created when you run:

```bash
cd app-front
npm run build
```

**Do not manually edit files in the `/build/` folder** - they are automatically generated.

### Folder Structure

```
assets/
├── DataBase.csv       # Your CSV data file (manual)
├── build/             # Auto-generated by Vite
│   ├── manifest.json  # Build manifest
│   ├── *.js          # Compiled JavaScript
│   └── *.css         # Compiled CSS
└── README.md         # This file
```
